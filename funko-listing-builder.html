Funko Pop! Vinyl

eBay Listing Builder

Search the offline catalog to pre-fill name/series/image, set the number & flags by hand, then get a ready-to-paste eBay description, title & item specifics.

About the data
The name/series autocomplete uses a bundled offline catalog (data/funko_pop.json) that is frozen at Jan 2021 — anything newer must be typed manually. There is no reliable Funko API, so Pop number, exclusivity and flags are always yours to enter. Live price comps come from eBay (/api/ebay) and are current asking prices, not sold. Set EBAY_APP_ID/EBAY_CERT_ID in .env to enable them.
Search above to pre-fill, or just type the fields. The preview always builds.
Box condition defaults to Near-Mint (not Mint) so an un-edited listing under-promises. A pro grade, if entered, becomes the headline condition. Only claim Chase/exclusive if the box truly says so — mis-stating it is the #1 cause of "not as described" disputes.
