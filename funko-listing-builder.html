Funko Pop! Vinyl

eBay Listing Builder

Search the offline catalog to pre-fill name/series/image, set the number & flags by hand, then get a ready-to-paste eBay description, title & item specifics.

About the data
The name search blends a bundled offline catalog (data/funko_pop.json, ~11k Pops frozen at Jan 2021) with live eBay results for anything newer — newer Pops carry an eBay tag and pre-fill a current photo + the Pop number parsed from the listing. There is no reliable Funko API, so exclusivity and flags are always yours to confirm. The live search and price comps both come from eBay (/api/ebay); prices are current asking, not sold. Set EBAY_APP_ID/EBAY_CERT_ID in .env to enable live results (offline search still works without them).
Search above to pre-fill, or just type the fields. The preview always builds.
Box condition defaults to Near-Mint (not Mint) so an un-edited listing under-promises. A pro grade, if entered, becomes the headline condition. Only claim Chase/exclusive if the box truly says so — mis-stating it is the #1 cause of "not as described" disputes.
