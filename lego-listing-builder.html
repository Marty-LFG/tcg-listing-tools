LEGO

eBay Listing Builder

Type a set number, get a ready-to-paste eBay description, title & item specifics. Data via Rebrickable + Brickset; new/used pricing via BrickLink.

API connection & keys
Lookups run through the dev-server proxies /api/lego/rebrickable (set data), /api/lego/brickset (RRP/age) and /api/lego/bricklink (new/used pricing). Add the matching keys to .env and run pnpm dev. Any missing key just disables that source — every field stays editable and the preview always builds.
Type a set number and hit Look up — or fill the fields by hand.
Condition defaults to the safest option (Used – Complete) so an un-edited listing never over-promises. Bare set numbers are auto-normalised to the -1 variant for lookup. BrickLink prices are real sold averages (new vs used); RRP comes from Brickset.
